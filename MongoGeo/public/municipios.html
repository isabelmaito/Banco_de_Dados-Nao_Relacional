    Sobre o Sistema

    O Sistema de Gestão Municipal é uma plataforma desenvolvida para facilitar o acesso e gerenciamento de informações sobre municípios brasileiros.

    Com esta ferramenta, você pode:

    • Consultar dados detalhados sobre todos os municípios do Brasil
    • Adicionar, editar e remover informações municipais
    • Encontrar municípios próximos à sua localização atual
    • Filtrar e ordenar municípios por diferentes critérios

    Esta aplicação utiliza uma API RESTful para gerenciar os dados, permitindo operações CRUD completas e consultas geoespaciais.
